# Mini Bank System 🏦

A simple console-based banking system built with Python.

This project simulates basic banking operations such as checking account balance, depositing money, and withdrawing money. It was created to practice Python fundamentals including loops, conditions, user input, and variables.

## Features

- Show current balance
- Deposit money
- Withdraw money
- Prevent withdrawing more than the available balance
- Simple menu-based user interface

## Technologies Used

- Python 3

## Concepts Practiced

- Variables
- While loops
- Conditional statements (if/elif/else)
- User input
- Basic error handling logic
